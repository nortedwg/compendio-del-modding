# Tools & Links Reference

Compiled by **NORTE.m2** · Version 1.0

## Blender

You'll need different versions depending on the task. Install them separately.

| Version | What it's for | Download |
|---|---|---|
| **Blender 4.3.2** | Importing Garry's Mod maps | [Download](https://download.blender.org/release/Blender4.3/) |
| **Blender 3.6 LTS** | Intermediate conversion step for GMod maps | [Download](https://www.blender.org/download/lts/3-6/#versions) |
| **Blender 3.4** | WMOs and M2s *(recommended)* | [Download](https://download.blender.org/release/Blender3.4/) |
| **Blender 2.90.0** | M2s *(original version of the M2i addon)* | [Download](https://download.blender.org/release/Blender2.90/) |

## Blender Addons

Install each one in the corresponding Blender version. *[[How to install an addon]](https://nortedwg.github.io/compendio-del-modding/WMO/Uso-basico-de-Blender-para-WMO#c%C3%B3mo-instalar-un-addon)* *[[Video]](https://youtu.be/q1Nvbl8oNpQ?si=7elpRkRGVa6TXrWL)*

| Addon | Blender Version | What it's for | Download |
|---|---|---|---|
| **SourceIO** | 4.3.2 | Importing Garry's Mod maps (`.bsp`) | [GitHub](https://github.com/REDxEYE/SourceIO) |
| **WoW: Useful Shortcuts** | 3.4 | WMO and M2 utilities. Highly recommended | [GitHub](https://github.com/nortedwg/WoW-Atajos-Utiles) |
| **WBS – Wow Blender Studio** | 3.4 | Exporting WMOs | [Drive](https://drive.google.com/file/d/11RFIqoz6zncmGiFw6dUGgpMvZR0iKJQ6/view) |
| **M2i Import** *(for 3.4)* | 3.4 | Import/export M2s in Blender 3.4 | [GitHub](https://github.com/nortedwg/m2i-blender-3.4) |
| **M2i Import** *(original 2.9)* | 2.90 | Import/export M2s in Blender 2.9 | [Bitbucket](https://bitbucket.org/suncurio/blender-m2i-scripts/src/master/) |
| **QuickSnap** | Any | Snap objects vertex-to-vertex | [GitHub](https://github.com/JulienHeijmans/quicksnap/archive/refs/heads/main.zip) |
| **Recommended Keybindings** | 3.4 | Pre-configured keymap profile | [Drive](https://drive.google.com/file/d/11Uwwvr1dN4LGdEEPyAjzp34we4XogzUR/view) |
